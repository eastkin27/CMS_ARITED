# CMS_ARITED

CREATION D'UN CMS POUR L'ARITED, PAR LE GROUPE DE 'LES SURVIVANTS' DE PARIS NANTERRE UNIVERSITÉ. 

OBJECTIF : 

-FOURNIR UN SYSTÈME DE GESTION DE CONTENU AUTONOME ET PROFESSIONNEL
-PERMETTRE LA CRÉATION ET L'ÉDITION EN TEMPS RÉEL D'ACTUALITÉS DE PROJETS ET DE PAGES STATIQUES 
-ASSURER LA TRANSPARENCE VIA L'AFFICHAGE DE L'AVANCEMENT DES PROJETS AVEC UNE BARRE DE PROGRESSION GÉRER 
-SUIVRE LES DEMANDES DE SERVICES CLIENTS INTRANTS DE RENDEZ-VOUS
-SIGNALEMENTS AVEC UN SYSTÈME DE STATUT NOUVEAU EN COURS TRAITÉ
-SÉPARER L'INTERFACE PUBLIQUE FRONTEND DE L'INTERFACE D'ADMINISTRATION BACKOFFICE POUR UNE GESTION CLAIRE
-UTILISER FIREBASE FIRESTORE POUR UN STOCKAGE DES DONNÉES SÉCURISÉ ET EN TEMPS RÉEL GARANTIR LA FONCTIONNALITÉ MULTISITE MULTITENANCY VIA UN FILTRE SITEID POUR GÉRER PLUSIEURS ENTITÉS -AVEC UNE SEULE APPLICATION ÊTRE 100% DÉPLOYABLE ET AUTONOME SUR GITHUB SOUS LE NOM DE VOTRE ENTREPRISE SANS DÉPENDANCE EXTÉRIEURE 
-UTILISER UNE PILE TECHNOLOGIQUE MODERNE ET MAINTENABLE REACT TAILWIND CSS
